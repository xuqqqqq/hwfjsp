# 1. 输入文件格式 

字段说明：

<table>
  <thead>
    <tr>
      <th>一级字段</th>
      <th>属性</th>
      <th>描述</th>
      <th>类型</th>
    </tr>
  </thead>
  <tbody>
    <tr><td rowspan="2">time</td><td>current_time</td><td>调度开始时间(单位：分钟)</td><td>int</td></tr>
    <tr><td>current_date_time</td><td>按照‘年/月/ 日 时: 分: 秒'表示的调度开始时间</td><td>str</td></tr>
    <tr><td rowspan="2">config</td><td>cal_threads</td><td>求解线程数(默认32)</td><td>int</td></tr>
    <tr><td>max_output_horizon</td><td>最大化限定时间内的产出(单位：分钟)</td><td>int</td></tr>
    <tr><td rowspan="4">eqp</td><td>eqp_id</td><td>设备 id 编号(设备唯一性标识)</td><td>str</td></tr>
    <tr><td>is_batch_type</td><td>是否属于并行组批加工设备</td><td>bool</td></tr>
    <tr><td>factory_info</td><td>设备所属厂区</td><td>str</td></tr>
    <tr><td>eqp_down_interval</td><td>设备维修时间段，按照两层列表格式[[begin1, end1]，[begin2, end2]]给出所有设备维修的开始结束时段</td><td>list</td></tr>
    <tr><td rowspan="25">task</td><td>task_id</td><td>待排任务id编号(任务唯一性标识)</td><td>str</td></tr>
    <tr><td>earliest_ava_time</td><td>任务最早可加工时间</td><td>int</td></tr>
    <tr><td>task_delivery_time</td><td>任务交期时间</td><td>int</td></tr>
    <tr><td>task_priority</td><td>任务优先级权重(数值越小，优先级越高)</td><td>float</td></tr>
    <tr><td>final_product_weight</td><td>任务成品产出重量</td><td>float</td></tr>
    <tr><td>process_path</td><td>任务下面可选的加工路径</td><td>dict</td></tr> 
    <tr><td>path_id</td><td>任务下面可选的加工路径编号</td><td>str</td></tr>
    <tr><td>qtime_info</td><td>对应的path_id下面隶属于同一task不同工序之间的最大最小加工间隔信息</td><td>dict</td></tr>
    <tr><td>qtime_id</td><td>最大/最小间隔参数编号</td><td>str</td></tr>
    <tr><td>start_process_seq</td><td>最大/最小间隔对应的起点工序编号</td><td>str</td></tr>
    <tr><td>start_process_type</td><td>起点工序加工间隔开始计时类型(取值为start/end,分别表示以当前工序的开工时间/完工时间计算加工间隔)</td><td>str</td></tr>
    <tr><td>end_process_seq</td><td>最大/最小间隔对应的终点工序编号</td><td>str</td></tr>
    <tr><td>end_process_type</td><td>终点工序加工间隔开始计时类型(取值为start/end,分别表示以当前工序的开工时间/完工时间计算加工间隔)</td><td>str</td></tr>
    <tr><td>min_process_interval</td><td>最小加工间隔时长，为None表示无此约束</td><td>int</td></tr>
    <tr><td>max_process_interval</td><td>最大加工间隔时长，为None表示无此约束</td><td>int</td></tr>
    <tr><td>process_list</td><td>对应的 `path_id` 下面的待加工工序列表</td><td>dict</td></tr>
    <tr><td>process_seq</td><td>加工路径下面加工工序的编号</td><td>str</td></tr>
    <tr><td>is_batch_type</td><td>当前工序是否为并行组批加工工序，为TRUE表示是并行组批加工工序</td><td>bool</td></tr>
    <tr><td>batch_family</td><td>当前工序的所属的工序family(仅 `is_batch_type` 属性为 true 的工序有此属性)，对于在同一批次加工的工序，要求这些工序的batch_family属性下面的 list 中每个位置上的属性都能相互模糊匹配上</td><td>list</td></tr>
    <tr><td>diff_factory_info</td><td>表示当前工序和下一工序之间允许发生跨厂转运的配置信息，此项为空表示不允许两个工序的加工发生转运，按照两层列表格式[[from_factory, to_factory], ...]的格式给出所有允许发生跨厂转运的配置</td><td>list</td></tr>
    <tr><td>eqp_list</td><td>当前工序加工可选机器列表</td><td>dict</td></tr>
    <tr><td>eqp_id</td><td>可选加工机器编号</td><td>str</td></tr>
    <tr><td>priority</td><td>可选加工机器优先级(数值越小优先级越高)</td><td>float</td></tr>
    <tr><td>process_time</td><td>可选加工机器加工当前工序所需加工时长</td><td>int</td></tr>
    <tr><td>curr_batch_size</td><td>如果当前工序未批次加工，当前机器加工当前任务同一批最多同时加工的数量</td><td>float</td></tr>
    <tr><td rowspan="3">transition</td><td>from_eqp</td><td>转运起点机器编号</td><td>str</td></tr>
    <tr><td>to_eqp</td><td>转运终点机器编号</td><td>str</td></tr>
    <tr><td>transition_time</td><td>转运时长(查不到的表示转运时长为 0)</td><td>int</td></tr>
    <tr><td rowspan="3">setup</td><td>from_process</td><td>上一工序编号，工序编号按照"(task_id, path_id, process_seq)"的格式表示，例如"('NQ0892', '0', '1')" 表示待排任务编号为'NQ0892'，路径编号'0'下面的工序'1'</td><td>str</td></tr>
    <tr><td>to_process</td><td>下一工序编号，编号格式同上</td><td>str</td></tr>
    <tr><td>setup_time</td><td>from_process-&gt;to_process在同一机器上紧邻加工所需的生产切换时长(查不到的表示切换时长为 0)</td><td>int</td></tr>
  </tbody>
</table>

# 2. 输出文件格式

字段说明：

| 属性名              | 描述                                              | 数据类型 |
| ------------------- | ------------------------------------------------- | -------- |
| task_id             | 待排任务id编号                                    | str      |
| process_seq         | 加工工序编号                                      | str      |
| temp_machine_id     | 当前工序选择的加工机器编号                        | str      |
| path_id             | 当前任务选择的加工路径编号                        | str      |
| process_start_time  | 按照‘年/月/ 日 时: 分:  秒'表示的当前工序开工时间 | str      |
| process_finish_time | 按照‘年/月/ 日 时: 分:  秒'表示的当前工序完工时间 | str      |

注：日期字符串可以用'-'或者'/'进行分隔。
